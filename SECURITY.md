# Security Policy

## Supported Versions

TerminAI is early-stage software. Security fixes are applied to the current
`main` branch and released as new versions. Older commits are not patched.

## Reporting a Vulnerability

Report security issues directly to security@billybox.dev rather than opening
a public issue.

Please include:
- Affected version or commit SHA
- Steps to reproduce
- Observed impact
- Suggested remediation, if any

Reports are triaged within a reasonable timeframe. If the issue is accepted,
a fix is prepared, tested through the normal release process, and disclosed
with credit unless you prefer anonymity.
